# SE_19 Personal Blog & Product Catalog

A full-stack Flask web application featuring a dynamic blog system and a static product showcase, developed for the Web Development Basics 2026 module.

## 🛠 Features
- **Dynamic Blog (CRUD):** Fully functional Create, Read, Update, and Delete operations for blog posts.
- **Secure Admin Panel:** Protected by a password-gate using Flask sessions and environment variables.
- **NoSQL Integration:** Connected to MongoDB Atlas for cloud-hosted data persistence.
- **Modular Templates:** Built with Jinja2 inheritance and partials for clean, maintainable HTML.
- **Responsive UI:** CSS-driven responsive design that adapts to mobile, tablet, and desktop screens.

## 🧰 Tech Stack
- **Backend:** Python / Flask
- **Database:** MongoDB Atlas (NoSQL)
- **ODM:** MongoEngine
- **Deployment:** Render / Gunicorn[cite: 2]
- **Frontend:** HTML5, CSS3, Jinja2[cite: 2]
